API Gateway Documentation

All endpoints require a valid X-API-Key header to access. You may also be rate-limited if you exceed 10 requests per minute.

GET /api/users

Header Required: X-API-Key

Description: Returns a list of users.

Success Response:

{
  "users": [
    { "id": 1, "name": "Alice" },
    { "id": 2, "name": "Bob" }
  ]
}

Error Responses:

  • 401 Unauthorized: Missing or invalid API key
  • 429 Too Many Requests: Rate limit exceeded

GET /api/products

Header Required: X-API-Key

Description: Returns a list of products.

Success Response:

{
  "products": [
    { "id": 101, "name": "Widget" },
    { "id": 102, "name": "Gadget" }
  ]
}

Error Responses:

  • 401 Unauthorized: Missing or invalid API key
  • 429 Too Many Requests: Rate limit exceeded
